HOWTO - Manage Books Add New Book in the bookshelf The dialogue provided for you to add your books in Booxar bookshelf. After successful entry, newly added book will be made available for others to borrow. You will start seeing it in the 'Bookshelf' tab in the app. My books - as Book Owner Dropdown list containing all the books logged in user has added in the bookshelf as 'Book Owner'. Books, for which the logged in user is not the owner, will not be listed. User can filter the books as per the current status - 'Available', 'Borrowed', 'Reserved'. Take Action: As a book owner you can perform following actions on the book. 1. Lend or Make Available : Scan the QR code presented by the book requester. If the QR code matches, requester or bnorrower authorization is successful. Lend the book to the borrower OR accept the book back from the borrower. 2. Enanble for AR (Augmented Reality): The dialogues provided here enable the book owner to add Anchors and corresponding content to be augmented. Anchors: Anchors are the makrers on top of which additional contextual content is augmented. i.e. when the app user points smart device on the anchors, augmented content is rendered. For the Booxar app, Anchor can be existing images in the book. Content: Conten is the media content you want to attach and upload so that it is augmented on the existing anchors in the book. When the app user points smart device on the anchors, this content is rendered. Once Anchor and Content are uploaded for the book, the book is 'AR Enabled' and it starts appearing in the ARBooks tab of the app. 3. Delete: The dialogue provided for the book owner to remove the book from bookshelf. Once the book is removed, other app users will not see this book listed in the Bookshelf tab. My books - as Book Requester The dialogue provided for you to view the books you have reserved or borrowed from other book owners. If the book has been borrowed, it also lists the 'Return By' date.
